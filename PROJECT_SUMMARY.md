## 🚀 Cara Memberikan ke Developer

1. **Kirim file ZIP** `indonesia-luxe-laravel.zip`
2. **Kirim juga** `DEVELOPER_GUIDE.md` untuk panduan lengkap
3. Developer cukup:
   ```bash
   unzip indonesia-luxe-laravel.zip
   cd indonesia-luxe-laravel
   composer install
   npm install
   cp .env.example .env
   php artisan key:generate
   # Setup database di .env
   php artisan migrate
   php artisan db:seed
   php artisan serve
   ```
